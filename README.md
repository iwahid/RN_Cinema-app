# Проект React Native Cinema-app

### Цель: 
#### Создать приложение для кинотеатра, которое позволило бы пользователям выбрать, забронировать и купить билеты на интересующие их сеансы

### Задача: 
#### Разработать приложение, в котором пользователи могут просмотреть актуальные и планируемые сеансы и фильмы. Получить детальную информацию о выбранном фильме. Выбрать подходящий сеанс, увидеть текущую загруженность зрительного зала на текущий момент, выбрать место, узнать цены и купить билет.

### Требования к приложению:
#### Для кинотеатра:
- [x] Лёгкий менеджмент сеансов, благодаря облегчённому процессу заполнения киноафиш. Для создания афиши, администратору достаточно указать в системе лишь уникальный id фильма, взяв его с Кинопоиска. После чего, вся необходимая информация о фильме будет получена со стороннего ресурса. 
- [x] Автоматизация бронирования и продажи мест. Нет необходимости в дополнительном ПО. Бронирование и продажа мест в одном приложении. 
- [x] Возможность в любой момент времени оценить загруженность зрительного зала и не допустить его перезаполнения. (К сожалению, крайне актуальное требование в текущий период)

#### Для пользователя:
- [x] Возможность отдельно просмотреть актуальные и планируемые сеансы в кинотеатре: фильмы, которые уже идут в прокате и только планируются в будущем.
- [x] Просмотр детальной информации о каждом фильме, представленном в прокате 
- [x] Возможность выбрать любой из доступных сеансов (как по дате, так и по времени)
- [x] Оценить в реальном времени текущую загруженность кинозала, просмотрев карту с проданными, на текущий момент, местами
- [x] Самостоятельно выбрать место в зале, ориентируясь на интерактивную схему зала
- [x] Возможность купить билет на выбранный сеанс


### Принцип работы приложения:
1. После запуска, приложение получает от кинотеатра список всех доступных к просмотру фильмов (только уникальные идентификаторы)
2. На основе полученного списка, загружает дополнительные данные по API с Кинопоиска и формирует афиши для доступных сеансов
3. При переходе к выбору сеанса (даты, времени и места просмотра фильма), приложение загружает актуальные данные о:
 - доступных датах показа
 - доступных временных сеансах
 - проданных на текущий момент билетах
 - карте зала и проданных в нём билетах
 - о ценах на каждый из сеансов
4. При просмотре и выборе пользователем интересующего его сеанса - приложение отображает актуальные для этого сеанса данные (доступность, стоимость)
5. Покупка билета через приложение. на текущий момент будет представлен в виде мокового модуля. (платить некому)
